# Netflix Clone

A responsive Netflix Clone web application built using HTML, CSS, and JavaScript. This project recreates the Netflix user interface with modern styling and interactive functionality for learning purposes.

## Features
- Responsive Netflix-inspired user interface
- Movie and series banners
- Interactive navigation bar
- Hover effects and smooth transitions
- Responsive design for multiple screen sizes
- JavaScript functionality

## Technologies Used
- HTML5
- CSS3
- JavaScript

## Assets Used
- Images and posters collected from online resources
- Icons and media assets customized for the project
- Netflix-inspired user interface design

## Project Purpose
This project was created to improve frontend development skills and understand responsive website development.

## Disclaimer
This project is for educational purposes only and is not intended for commercial use.
